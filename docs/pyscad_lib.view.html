pyscad_lib.view.view_models module

class pyscad_lib.view.view_models.SelectionView(list_node, list_elem, currentLoading, currentAction, currentFixedStep, currentRHS)

Базовые классы: object

Класс информации об объектах, выбранных пользователем, на момент вызова функции.

Параметры

  • list_node ([int]) – Массив номеров узлов

  • list_elem ([int]) – Массив номеров элементов

  • currentLoading (int) – Номер загружения для режима задания исходных данных (или – 0)

  • currentAction (int) – Номер воздействия для режима анализа результатов (или – 0)

  • currentFixedStep (int) – Номер шага для режима анализа результатов (или – 0)

  • currentRHS (int) – Номер правой части для режима анализа результатов (или – 0)
